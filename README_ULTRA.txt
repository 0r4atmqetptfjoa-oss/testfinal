
# Ultra‑Modern Visual Enhancements Pack

Include:
- **Spotlight** (cursor‑reactive radial gradients, subtil pe fundal)
- **TiltCard** (parallax tilt 3D + glow border)
- **ShimmerButton** (buton cu efect de shine)
- **AnimatedCounter** (numere care cresc fluid)
- **Toast** (notificări XP)
- **ProgressBar** (accent violet/cyan)
- **WeeklyBar/Stat** (widgeturi)
- **MotionProvider** (respectă `prefers-reduced-motion` global)
- **Confetti** (canvas) pentru finalizare & badge-uri
- **Home/QuizPlayer/Progress** refactor cu micro‑interacțiuni moderne

> Paletă: `bg-black`, card `bg-gray-900`, borduri `border-gray-800`, text `text-gray-200/500`, accente `violet-500` & `cyan-400`.

## Dependențe
```bash
npm i framer-motion recharts lucide-react
```

## Integrare
- Copiază fișierele în proiect (peste `src/`).
- Adaugă `<MotionProvider>` la nivelul de sus al aplicației (opțional) sau păstrează local utilizarea componentelor.
- Router: asigură ruta către `Progress` dacă nu există.
